QuietPage OG Image Generator

This HTML template generates the Open Graph image at 1200x630px.

To create the PNG:

1. Open this file in a browser

2. Use browser DevTools to screenshot the .og-image element

3. Or use a tool like Puppeteer/Playwright to capture programmatically

4. Save as og-image.png in the same directory
